# Manual do Usuário
### Login

- Ao acessar o sistema, informe usuário e senha.

### Cadastro de livros

- Clique em "Novo Livro".

- Preencha:

    - título
    - autor
    - ISBN

- Clique em Salvar.

### Pesquisa

- Utilize a barra de pesquisa para localizar livros.

### Empréstimos

- Selecione um livro disponível.

- Escolha o usuário.

- Clique em Confirmar.

### Devolução

- Selecione o empréstimo.

- Clique em Registrar devolução.

### Encerramento

- Utilize o botão Sair para finalizar sua sessão.
