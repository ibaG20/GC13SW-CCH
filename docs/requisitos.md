# Documento de Requisitos

## Objetivo

O sistema BookFlow tem como objetivo facilitar o gerenciamento de bibliotecas, permitindo controlar livros, usuários e empréstimos.

## Requisitos Funcionais

- RF01 – Cadastrar livros
- RF02 – Editar livros
- RF03 – Excluir livros
- RF04 – Pesquisar livros
- RF05 – Cadastrar usuários
- RF06 – Registrar empréstimos
- RF07 – Registrar devoluções
- Requisitos Não Funcionais
- RNF01 – Interface simples.
- RNF02 – Sistema responsivo.
- RNF03 – Dados armazenados em banco de dados.
- RNF04 – Apenas usuários autenticados podem acessar o sistema.

## Restrições

- O sistema será acessado via navegador.
